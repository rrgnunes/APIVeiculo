## CEABS api Veiculo

`Git Clone`
```
git clone https://github.com/rrgnunes/APIVeiculo.git
```
`Change Directory`
```
cd APIVeiculo
```

`Restore Packages`
```
dotnet restore
```

`Build Command`
```
dotnet build
```

`Run Command`
```
dotnet run
```

`Watch Command`
```
dotnet watch run
```

`Check Url`

http://localhost:5000/swagger

https://localhost:5001/swagger

```
dotnet add package Microsoft.EntityFrameworkCore
```
